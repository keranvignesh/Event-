READY PLAYER ONE
Description Of Event :

Description Of First Round:

  • The Qr Code Will Be Given To The Player.
  • After Scanning The Qr Code There Is A Pdf Which Contains The 10 Hash Files For The Single Player And The 15 Hash Files For The Group Players .
  • Out Of Given Hash Files The Player Has To Select Any 4 Hash Files To Decrypt The Given Text File
  • If The File Is Decrypted, The Player Will Get The Link To Move To The Next Round.

Description Of Second Round:

  • This Round Consists Of Code Or The Stenography Images.
  • The Player Has To Decode The Code Or Stenography Images To Get The Link To The Next Round That Is Mathemetical Riddle.

Description Of Third Round:

  • After Solving The Mathematical Riddle The Player Will Win Out The Game

Rules

  • Individual And Group Participation(Minimum 2 Members).
  • 4 Questions In Different Difficulty Levels.
  • Computers Will Be Provided
  • Time Limit Is 90 Min.
Time Duration: 1 hour
