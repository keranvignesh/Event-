CHANNEL SURFING
Description Of Event :

 Channel surfing is the action or practice of surfing through television programs usually by use of a remote control. Channel Surfing offers audiences a varied exhibition representing themes of devotion, tenderness, romance, tragedy, celebration

Rules:

  • It’s a group event

  • Team contains 5 to 7 members

  • Each team can perform 5-7 minutes

  • Channels will be given by judges on-spot

  • It’s a single round event

  • Only 1 winner & 1 runner will be awarded

Immediate DQ:

  •  using vulgarity & foul languages

  •  using adults contents

  • Using of college name.

  • Ads not allowed


Criteria For Evaluation:

  • Performance

  • Spontaneity

  • Fun elements

  • Team co-ordination

         *Winners will be selected by the judges  based on the criteria.

Time duration: 3 hour
Number Of Rounds : 1
